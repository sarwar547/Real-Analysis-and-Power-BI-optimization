# 🛒 Sales E-Commerce Dashboard — Power BI

An end-to-end interactive business intelligence dashboard built using **Power BI**, 
designed to analyze e-commerce sales data and deliver actionable insights across 
revenue, customers, products, and store performance.

---

## 📊 Dashboard Overview

This project presents a comprehensive sales analytics solution for an e-commerce 
business, enabling stakeholders to monitor KPIs, identify trends, and make 
data-driven decisions through dynamic visualizations.

---

## 🔍 Key Features

- 📈 **Revenue & Sales Trends** — Monthly and yearly sales performance tracking
- 👥 **Customer Behavior Analysis** — Cohort analysis, retention, and segmentation
- 🛍️ **Product Performance** — Best-selling products, category-wise breakdown
- 🏪 **Store Profitability** — Location-wise and channel-wise profit analysis
- ⚡ **Automated ETL Pipeline** — Data cleaned and transformed using SQL & Python
- 🎯 **Dynamic Filters** — Slicers for date range, region, category, and more

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
|------|---------|
| Power BI | Dashboard & Visualizations |
| SQL | Data Modeling & ETL (CTEs, Window Functions, Joins) |
| Python (Pandas) | Data Cleaning & Preprocessing |
| DAX | Calculated Measures & KPIs |
| Excel / CSV | Raw Data Source |
